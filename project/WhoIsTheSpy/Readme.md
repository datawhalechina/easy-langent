# WhoIsTheSpy

## 项目简介

本项目是《第八章 综合实践：构建“谁是卧底”游戏智能体》的实践作品。项目使用 LangChain 和 LangGraph 实现一个“谁是卧底”多智能体游戏引擎，用户作为旁观者查看 4 个智能体自动完成发言、投票和胜负判断。

## 核心功能

- 自动生成一组“平民词”和“卧底词”
- 随机为 4 个智能体分配角色，其中 1 个卧底、3 个平民
- 智能体根据自己的词语生成发言，且不能直接说出词语
- 智能体根据所有发言进行投票
- 根据投票结果淘汰玩家，并自动判断胜负
- 使用 LangGraph 串联完整游戏流程

jupyter-env/bin/python --version

## 依赖安装

如果已经在仓库根目录创建并使用 `jupyter-env`，可以直接安装项目依赖：

```bash
jupyter-env/bin/pip install langchain langchain-openai langgraph python-dotenv
```

也可以使用项目根目录的 `pyproject.toml` 安装依赖。

## API Key 配置

在本目录下创建 `.env` 文件：

```bash
touch project/WhoIsTheSpy/.env
```

写入自己的 DeepSeek API Key：

```env
API_KEY=你的API_KEY
```

注意：`.env` 文件只用于本地运行，不能提交到 GitHub，避免 API Key 泄露。

## 运行步骤

在仓库根目录执行：

```bash
jupyter-env/bin/python project/WhoIsTheSpy/WhoIsTheSpy.py
```

如果本机配置了 `ALL_PROXY=socks://...`，并遇到 `Unknown scheme for proxy URL` 报错，可以临时取消代理后运行：

```bash
env -u ALL_PROXY -u all_proxy -u HTTP_PROXY -u http_proxy -u HTTPS_PROXY -u https_proxy \
  jupyter-env/bin/python project/WhoIsTheSpy/WhoIsTheSpy.py
```
